## 📊 Metodologias de Trabalho

* **Test-Driven Development (TDD) em ML:** Implementação de testes unitários para pipelines de dados (`Pytest`), garantindo que quebras de schema não cheguem à produção.
* **Criptografia e Privacidade:** Aplicação de mascaramento de dados sensíveis e autenticação segura baseada em chaves criptográficas (`API Keys/Secrets`).
* **MLOps e Reprodutibilidade:** Ambientes 100% isolados e replicáveis utilizando `Docker` e controle de versões de artefatos (`Joblib/Parquet`).
